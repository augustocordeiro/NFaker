FFaker.net
==========

FFaker.net is a rewrite of ruby [ffaker](http://rubygems.org/gems/ffaker) .

## Usage

```csharp
using FFaker
Faker.Name.name => "Christophe Bartell"
Faker.Internet.email => "kirsten.greenholt@corkeryfisher.info"
```
